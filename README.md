# Overview #
.Net API for a point of sales and inventory solution for Pom and Honey as part of Texas A&M's CSCE 331 course.

# Usage #

**To Install Dependencies**
```bash
dotnet restore
```

**To Run Project**
```bash
dotnet run --project .\Project2Api\
```

**To Test Project**
```
dotnet test
```
